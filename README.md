# Assignment 
Use Agile principles with a team to create a mobile-responsive front-end web application with JavaScript that incorporates the use of two different Web APIs. (10.1)

## Background
Students will create a 3-page website using JavaScript. This group project is meant to expose students to working in teams using Agile/Scrum techniques to meet project requirements and deadlines. Students will also gain practice using Git and GitHub in a team environment.

Students will be assigned to groups by the instructor before module 10.


## Project Requirements
To receive full credit for this assignment, the finished group project website MUST:

Be at least three pages.
Utilize at least two different external web APIs.
Be responsive on mobile, tablet, and desktop devices.
Utilize localstorage to persist data retrieved from the user or the external APIs in the browser.
In addition to satisfying the above requirements, ALL project members must contribute to the JavaScript written for the site in order to receive credit. Group members MUST also use GitHub issues to track features they work on to complete the project.
